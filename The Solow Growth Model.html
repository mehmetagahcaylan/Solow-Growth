The Solow Growth Model


The Solow Growth Model is a macroeconomic growth projection method with three main variables:
K indicates the capital stock. L is the total labor. In this research the starting populations of all countries were presumed equal as one million.

For example, the sewing machine in a textile factory is a part of the capital stock and the person operating the machine is a unit of labor. In this research, we will include the total factor productivity which is shown by A.

Contemporary research always includes Human Capital as a variable. The idea of increasing human productiveness and its macroeconomic effects were put forth by Gary Becker. This research will not include the human capital. Nonetheless, the human capital is regarded as the most important variable in calculating the growth of developed countries, this research must be read taking this into consideration.

Parameters of the Solow Model


Q Output
A Productivity Parameter
K Capital Stock
L Labor / million
I Investment
S Savings

simulation <- function(Q, K, L, A, I, S) {
  for (i in c(1:55)) {
    s <- 0.15 #Saving
    d <- 0.25 #Depreciation
    n <- 0.02 #Population
    a <- 0.4 #Output elasticity of capital
    y <- 0.01 #TFP growth rate
    t <- 55 #Time (year)
    ss <- Q[i]*s
    ii <- s*Q[i]
    ll <- (1+n)*L[i]
    aa <- (1+y)*A[i]
    kk <- I[i]+(1-d)*K[i]
    qq <- A[i]*(K[i]^a)*(L[i]^(1-a))
    S[i+1] <- ss
    I[i+1] <- ii
    L[i+1] <- ll
    A[i+1] <- aa
    K[i+1] <- kk
    Q[i+1] <- qq
  }
  dt <- data.table(year = c(1:56), Q=Q, I=I, L=L, K=K, S=S, A=A)
  return(dt)
}
simulation_function <- function(data, title, subtitle) {
  g1 <- ggplot(data, aes(x = year)) +
    geom_line(aes(y=S, color = "Savings"), size = 1.5) +
    geom_line(aes(y=I, color = "Investment"), size = 1.5) +
    geom_line(aes(y=Q, color = "GDP"), size = 1.5) +
    geom_line(aes(y=K, color = "Capital Stock"), size = 1.5) +
    labs(title = title, subtitle = subtitle,
         y = "Total Amount (dollars)", x = "Years") +
    theme_bw() +
    scale_color_manual(name = NULL,
                       values= c("GDP" = "blue",
                                 "Investment" = "red",
                                 "Capital Stock" = "black",
                                 "Savings" = "orange"))
  return(g1)
}
